## 🔬 Featured Medical AI Project

### 👁️ Glaucoma-Detection-HPC
**Advanced Quality-Aware Deep Learning Diagnostics Pipeline on the HYGD Dataset**
* Built a state-of-the-art heterogeneous ensemble (**EfficientNet-V2**, **Swin Transformer**, and **ConvNeXt-V2**) to optimize local and global retinal features.
* Engineered a custom **Quality-Weighted Focal Loss (QWFL)** to dynamically down-weight bad/blurry fundus photos without data reduction.
* Integrated **Monte Carlo (MC) Dropout** for clinical safety, triggering uncertainty alerts to clinicians when the standard deviation of inference passes spikes.
* Utilized **GradCAM++** and attention rollouts to validate explainability across the Optic Nerve Head.

* **Tech Stack:** PyTorch, Albumentations, EfficientNet, Vision Transformers, MC Dropout, Slurm/HPC
